# xmedal.tv

An embed fixer for Medal inspired by [InstaFix](https://github.com/Wikidepia/InstaFix) and [FxEmbed](https://github.com/FxEmbed/FxEmbed).

Hosted on [Railway](https://railway.com?referralCode=astrid) with replicas in US West & East, EU West, and SEA.

## How to use

### Embed media only

Simply prefix the medal.tv domain with the letter `x`.
`https://medal.tv/games/abc/clips/xyz` -> `https://xmedal.tv/games/abc/clips/xyz`
